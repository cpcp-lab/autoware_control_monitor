# Batch mode

## Requirements

Extend the monitoring tool to process a collection of single-run log directories in batch:

- When the `--batch` option is given, accept a parent directory as a CLI argument.
- Subdirectories are assumed to be named as `0/`, `1/`,..., `n/` or `1/`, `2/`,..., `n/`.
- Run the existing single-run analysis on each subdirectory.
- For each single-run, aggregate the statistics for the windows:
    - Number of windows that satisfy the conditions.
    - Number of windows that violate any of the conditions.
- Aggregate the per-run statistics and report the result.

## Basic design

### Refactoring

Extract the monitoring loop from `main()` into a standalone function so that batch mode can call it without triggering visualization.

**`SingleRunResult` dataclass**

| Field | Type | Description |
|-------|------|-------------|
| `per_window` | `list[dict]` | One entry per window: `{i, a1, a2, f1, f2, g1, gh, gl}` (int accumulators, same as current print values) |
| `valid_count` | `int` | Number of windows where all conditions passed (`a1&a2&f1&f2&g1&gh&gl == 1`) |
| `total_count` | `int` | Total number of windows evaluated |

**`BatchResult` dataclass**

| Field | Type | Description |
|-------|------|-------------|
| `runs` | `list[tuple[str, SingleRunResult]]` | `(subdir_name, result)` pairs in sorted order |
| `valid_total` | `int` | Sum of `valid_count` across all runs |
| `total` | `int` | Sum of `total_count` across all runs |

**`run_single(data_dir, params) -> SingleRunResult`**

Move the following out of `main()`:
- CSV loading and `merge_asof` synchronization
- Windowed sampling (`m_range`, `m_subset`)
- Per-window monitoring loop (all condition checks, accumulators)
- Computation of `valid_count` and `total_count`

The plotting code (Plot 1, Plot 2, Plot 3) remains in `main()` and receives the returned `SingleRunResult`.

**`main()` dispatch**

```
if args.batch:
    result = run_batch(args.data_dir, params)
    # print summaries (no matplotlib)
else:
    result = run_single(args.data_dir, params)
    # print per-window summaries (existing behaviour)
    # show plots
```

### CLI

Two modes controlled by a new `--batch` flag:

```
# Single-run (existing behaviour)
python3 main.py [--window W] [--debug] [params...] <data_dir>

# Batch mode (new)
python3 main.py --batch [--window W] [params...] <parent_dir>
```

In batch mode:
- Skip visualization (no matplotlib output).
- Print a one-line summary per subdirectory: `<subdir>: <valid_num>/<total_num>`.
- Print the overall result: `total: <valid_num_total>/<total_num_overall>`.
